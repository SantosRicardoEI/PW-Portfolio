# Fotografias do Making Of

Esta pasta destina-se exclusivamente a fotografias reais do caderno, DER e
outros apontamentos produzidos durante o trabalho.

Sugestão de nomes:

- `01-entidades-iniciais.jpg`
- `02-der-versao-1.jpg`
- `03-der-versao-2.jpg`
- `04-correcao-modelacao.jpg`

As imagens devem estar legíveis, sem dados pessoais desnecessários, e cada uma
deve receber uma legenda no modelo `EvidenciaMakingOf` através do Admin.

Não colocar nesta pasta imagens geradas para simular trabalho manuscrito.
