Sobre esta Aplicação

Arquitetura MVT

Diagrama

Componentes

  • Model — define as entidades e a estrutura da base de dados (models.py).
  • View — recebe os pedidos HTTP, consulta os modelos e retorna respostas (views.py).
  • Template — ficheiros HTML que renderizam os dados enviados pelas views.
  • URLs — mapeiam os caminhos HTTP para as funções view (urls.py).

Diagrama Entidade-Relação

Principais entidades

  • Licenciatura → tem várias UnidadesCurriculares e Docentes
  • Projeto → associado a UCs e Tecnologias
  • TFC → tem Alunos, Orientadores, Áreas e Tecnologias
  • Tecnologia → tem um TipoTecnologia
  • Competência → ligada a Projetos e Tecnologias
  • Formação → ligada a Competências
  • MakingOf → documenta o processo de desenvolvimento
